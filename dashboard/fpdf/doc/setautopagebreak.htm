SetAutoPageBreak

SetAutoPageBreak(boolean auto [, float margin])

Deskripsi

Mengaktifkan atau menonaktifkan penanda akhir halaman otomatis. Jika aktif, parameter kedua adalah jarak dari bawah halaman yang menunjukkan batas minimal agar penanda akhir halaman diberikan. Secara default, mode ini aktif dan margin diset 2 cm.

Parameter

auto
true jika aktif atau false jika tidak aktif.
margin
Jarak dari bawah halaman.

Lihat juga

Cell, MultiCell, AcceptPageBreak
