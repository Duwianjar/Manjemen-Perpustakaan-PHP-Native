Error

Error(string msg)

Deskripsi

Metod ini secara otomatis dipanggil jika ada kesalahan fatal; metod ini hanya akan menampilkan exception dengan pesan yang diberikan.
Class turunan dapat meng-override metod ini untuk menangani kesalahan tertentu akan tetapi metod ini tidak boleh memberikan nilai kembalian, jika tidak maka dokumen yang dihasilkan bisa korup.

Parameter

msg
Pesan kesalahan.
