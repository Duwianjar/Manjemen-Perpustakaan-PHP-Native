Header()

Deskripsi

Metode ini digunakan untuk menampilkan header. Metode ini dipanggil secara otomatis oleh AddPage() dan tidak disarankan untuk dipanggil langsung dalam aplikasi. Implementasi metode ini dalam FPDF adalah kosong, jadi Anda harus membuat subclass dan menimpanya untuk mendapatkan hasil sesuai keinginan.

Contoh

class PDF extends FPDF
{
function Header()
{
   //Pilih font Arial bold 15
   $this->SetFont('Arial','B',15);
   //Geser ke kanan
   $this->Cell(80);
   //Judul dalam bingkai
   $this->Cell(30,10,'Title',1,0,'C');
   //Ganti baris
   $this->Ln(20);
}
}
