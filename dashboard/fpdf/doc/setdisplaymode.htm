SetDisplayMode

SetDisplayMode(mixed zoom [, string layout])

Deskripsi

Menentukan bagaimana dokumen ditampilkan oleh viewer. Tingkatan perbesaran (zoom level) dapat diset: halaman dapat ditampilkan seluruhnya dalam layar, memenuhi lebar jendela (window), sesuai ukuran aslinya, dengan faktor skala atau menggunakan nilai default viewer (dapat diset melalui menu Preferences dalam Acrobat). Layout halaman juga dapat ditentukan: sekaligus, tampilan berlanjut (continuous display), dua kolom atau default viewer.
Secara default, dokumen ditampilkan menggunakan mode lebar halaman penuh dengan tampilan berlanjut.

Parameter

zoom
Nilai zoom yang digunakan. Nilainya adalah:
  • fullpage: menampilkan satu halaman penuh di layar
  • fullwidth: menampilkan dokumen sehingga lebarnya memenuhi jendela
  • real: menampilkan dokumen dengan ukuran sebenarnya (sama dengan nilai zoom 100%)
  • default: menggunakan nilai default viewer
atau angka yang menunjukkan nilai zoom.
layout
Layout halaman. Nilainya adalah:
  • single: menampilkan halaman per halaman
  • continuous: menampilkan halaman berkelanjutan
  • two: menampilkan dua halaman dalam dua kolom
  • default: menggunakan nilai default viewer
Nilai default adalah continuous.
