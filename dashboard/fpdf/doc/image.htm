Image

Image(string file [, float x [, float y [, float w [, float h [, string type [, mixed link]]]]]])

Deskripsi

Menampilkan gambar. Ukuran gambar yang ditampilkan dapat ditentukan dengan berbagai cara:
  • lebar dan tinggi ditentukan secara eksplisit (dinyatakan dalam satuan yang diberikan pada constructor atau DPI)
  • salah satu ukuran ditentukan secara eksplisit, yang lainnya akan dihitung secara otomatis untuk menjaga proporsional gambar aslinya
  • ukuran tidak ditentukan; dalam hal ini gambar akan ditampilkan dalam 96 dpi
Format gambar yang didukung adalah JPEG, PNG dan GIF. Ekstensi GD diperlukan untuk menampilkan GIF.

Untuk JPEG, semua kedalaman warna boleh:
  • skala abu-abu (gray scale)
  • true colors (24 bit)
  • CMYK (32 bit)
Untuk PNG, hanya:
  • gray scale maksimal 8 bit (256 warna)
  • warna indeks (indexed colors)
  • true colors (24 bit)
Untuk GIF: jika merupakan animasi, hanya gambar (frame) pertama yang digunakan.

Mendukung transparensi.

Format gambar dapat dinyatakan secara ekspisit atau secara otomatis ditentukan dari ekstensi file.

Anda dapat membuat link pada gambar.

Catatan: jika sebuah gambar digunakan berkali-kali, hanya satu salinan yang disertakan (embed) dalam dokumen.

Parameter

file
Lokasi (path) atau URL gambar.
x
Nilai absis pojok kiri atas. Jika tidak ditentukan atau nilainya sama dengan null, maka nilai absis sekarang yang akan digunakan.
y
Nilai ordinat pojok kiri atas. Jika tidak ditentukan atau nilainya sama dengan null, maka nilai ordinat sekarang yang akan digunakan; tambahan lagi, penanda akhir halaman akan dibuat terlebih dahulu jika perlu (jika pemberian penanda akhir halaman otomatis aktif) dan, setelah memanggil metode ini, nilai ordinat sekarang akan berpindah ke bagian bawah gambar.
w
Lebar gambar yang akan ditampilkan. Ada tiga pilihan:
  • Jika nilainya positif, maka menunjukkan lebar dalam unit yang dipilih
  • Jika nilainya negatif, maka nilai absolutnya menunjukkan resolusi horizontal dalam DPI
  • Jika tidak ditentukan atau sama dengan nol, maka akan dihitung secara otomatis
h
Tinggi gambar yang akan ditampilkan. Ada tiga pilihan:
  • Jika nilainya positif, maka menunjukkan tinggi dalam unit yang dipilih
  • Jika nilainya negatif, maka nilai absolutnya menunjukkan resolusi vertikal dalam DPI
  • Jika tidak ditentukan atau sama dengan nol, maka akan dihitung secara otomatis
type
Format gambar. Nilainya adalah (case insensitive): JPG, JPEG, PNG and GIF. Jika tidak dinyatakan secara ekspisit, maka otomatis ditentukan dari ekstensi file.
link
URL atau nilai identitas yang dikembalikan oleh AddLink().

Contoh

// Tampilkan logo pada sudut kiri atas pada 300 DPI
$pdf->Image('logo.png',10,10,-300);
// Tampilkan gambar dinamis dari URL
$pdf->Image('http://chart.googleapis.com/chart?cht=p3&chd=t:60,40&chs=250x100&chl=Hello|World',60,30,90,0,'PNG');
