Rect

Rect(float x, float y, float w, float h [, string style])

Deskripsi

Membuat persegi panjang. Persegi panjang yang dibuat bisa berupa bingkai saja, arsiran (tanpa bingkai) atau keduanya.

Parameter

x
Absis pojok kiri atas.
y
Ordinat pojok kiri atas.
w
Lebar.
h
Tinggi.
style
Tampilan persegi panjang yang akan dibuat. Pilihannya adalah:
  • D atau string kosong: draw (bingkai saja). Ini adalah nilai default.
  • F: fill (arsiran saja)
  • DF atau FD: bingkai dan arsiran

Lihat juga

SetLineWidth, SetDrawColor, SetFillColor
