Output

string Output([string dest [, string name [, boolean isUTF8]]])

Deskripsi

Mengirim dokumen ke tujuan yang ditentukan: browser, file atau string. Dalam hal dikirim ke browser, plug-in dapat digunakan (jika ada) atau download (kotak dialog "Save as") dapat ditampilkan.
Metode ini akan memanggil Close() terlebih dahulu jika perlu untuk mengakhiri dokumen.

Parameter

dest
Tujuan pengiriman dokumen. Nilainya adalah:
  • I: mengirim file langsung ke browser (inline). Plug-in akan digunakan jika ada.
  • D: mengirim file ke browser dan memaksa download file dengan nama yang diberikan oleh name.
  • F: menyimpan file lokal dengan nama yang diberikan oleh name (dapat menyertakan path).
  • S: mengembalikan dokumen sebagai string. name diabaikan.
Nilai awal adalah I.
name
Nama file. Akan diabaikan jika tujuan adalah S.
Nilai awal adalah doc.pdf.
isUTF8
Menunjukkan bahwa nama dienkod dalam ISO-8859-1 (false) atau UTF-8 (true). Hanya digunakan untuk tujuan I dan D.
Nilai awal adalah false.
