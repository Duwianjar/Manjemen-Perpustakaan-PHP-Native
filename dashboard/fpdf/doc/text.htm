Text

Text(float x, float y, string txt)

Deskripsi

Menampilkan teks. Posisi awalnya adalah di kiri bawah (baseline) karakter pertama. Metode ini memungkinkan untuk menampilkan teks secara tepat dalam halaman, akan tetapi biasanya lebih mudah menggunakan Cell(), MultiCell() atau Write() yang merupakan metode-metode standar untuk menampilkan teks.

Parameter

x
Nilai absis awal.
y
Nilai ordinat awal.
txt
Teks yang akan ditampilkan.

Lihat juga

SetFont, SetTextColor, Cell, MultiCell, Write
